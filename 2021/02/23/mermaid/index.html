Markdown之mermaid插件

2021-02-23 Markdown 325 字
markdown mermaid

mermaid插件画流程图、时序图(顺序图)、甘特图。

官网说明:https://mermaid-js.github.io/mermaid/#/


横向流程图

graph LR
A[方形] -->B(圆角)
    B --> C{条件a}
    C -->|a=1| D[结果1]
    C -->|a=2| E[结果2]
    F[横向流程图]


竖向流程图

graph TD
A[方形] --> B(圆角)
    B --> C{条件a}
    C --> |a=1| D[结果1]
    C --> |a=2| E[结果2]
    F[竖向流程图]
